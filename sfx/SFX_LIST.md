# Sound Effects Library — 50 sounds

Drop files into this `sfx/` folder named **exactly** as below (`.mp3` preferred; `.wav`, `.ogg`, `.flac`, `.m4a` also work).
Example: `whoosh_fast.mp3`

The AI Director picks from these names automatically when planning beats. Missing files are skipped at render time (no errors). Keep clips **short (0.3–2s)** except risers/applause (2–4s).

Good free sources: pixabay.com/sound-effects, freesound.org, mixkit.co (check licenses).
