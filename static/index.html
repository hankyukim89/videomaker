AI Director
One pass plans the whole video. It splits the script two ways: voice segments (below — how the narration is read aloud, one audio file each) and the image plan (in the Images tab — which picture is on screen at which moment). They're separate so a picture can change mid-sentence without affecting the voice. It also writes the continuity bible, title, thumbnail prompt and chapters.
